## Installation and launch
**Cloning the slokaln repository project**
```Bash
git clone https://github.com/DKMFzF/bad-apple-node.js.git
```

**Going to the project directory**
```Bash
cd bad-apple-node.js
```

**Locking up third-party libraries**
```Bash
npm install
npm run start
```

or

```Bash
yarn
yarn start
```

## Assembly
